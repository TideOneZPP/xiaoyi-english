打开网址，直接进入书架

不需要准备教材包，也不需要选择任何本地文件。

Safari 打开“小译同学”

打开左上角书架

按需缓存课页

已经打开过的页面会自动缓存；首次打开新页面时仍需要网络。

选择课本并开始学习

建议每次 10—15 分钟，先读课文，再做一类练习。

① 打开左上角目录

在 iPad 上点击左上角三条横线，选择年级和上下册。

② 选择正在学习的 Unit

目录显示 64 个已核对的单元标题和起始页。

③ 直接翻课本页面

用左右箭头或输入页码；页面在 iPad 横竖屏都会自动适配。

④ 点黄色英文区域

点一句读一句；点“点读提示”可显示或隐藏黄色热区。

常见问题

遇到问题时先从这里检查。

还需要导入教材包吗？

不需要。8 册教材已经接入静态页面，打开书架直接选择即可。

还需要电脑吗？

不需要。iPad 直接访问 GitHub Pages，电脑无需开机，也无需连接同一个 Wi-Fi。

为什么某个新页面在断网时打不开？

教材按页加载。只有曾经打开并成功缓存的页面能离线使用；请恢复网络后重新打开该页。

为什么读音与出版社录音略有不同？

工具调用 iPad 系统的英式英语合成语音，不是教材出版社的真人配套音频，因此音色、连读和语调可能不同。

为什么有少量黄色框位置或文字不完全准确？

黄色文字层由 OCR 自动识别，复杂插图页可能出现漏字或错位。页面图片本身不受影响，可关闭“点读提示”正常阅读。

答错怎么办？

练习不会扣分。查看题目所标的课本页码，翻回原页核对后再试。

为什么不能自动判断孩子发音？

当前跟读采用孩子自我确认，不录音、不上传，也不伪装成自动评分。家长可在旁边听一遍并鼓励纠正。

五、六年级为什么标记为“官方平台现行版”？

三、四年级 PDF 标明“根据 2022 年版课程标准修订”；五、六年级使用本地已下载并核验的国家平台当前版本，界面不自行冒充尚未取得的新版。
