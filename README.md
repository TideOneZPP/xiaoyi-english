# 小译同学 · 英语同步练

给家里小学生使用的译林版英语同步练习工具。当前内置三至六年级共 8 册、64 个单元的原创同步练习，包含核心词汇、词义辨析、听力、句型补全、单词排序和情景/语篇理解。

每个单元另有“同步点读”：孩子打开即可逐词逐句点扬声器播放，也可长按选中任意英文后点击“朗读选中内容”，不需要家长填写。

每册均提供“国家平台正版教材”入口，跳转国家中小学智慧教育平台的对应电子教材；平台要求登录后查看正文。本项目不复制或绕过登录获取教材原文。

三、四年级采用当前已发布的 2024 新版目录；五、六年级在对应新版正式发布前采用学校仍在使用的现行版。详细审核记录见 `CONTENT_REVIEW.md`。

## 家庭本地使用

双击 `启动学习.bat`。首次启动会自动安装本地依赖，随后浏览器会打开学习页面。命令窗口需要保持打开；学习结束后直接关闭即可。

- 不需要公网部署
- 不需要账号或登录
- 不接后端数据库
- 不上传孩子的学习数据
- 学习进度仅存储在当前浏览器的 `localStorage` 中

语音播放使用 Windows/浏览器自带的英文语音能力。如果系统没有英文语音包，页面仍可完成文字练习。

## 在 iPad 上使用

1. 用 iPad Safari 打开发布后的学习网址。
2. 点 Safari 的“共享”按钮。
3. 选择“添加到主屏幕”。
4. 以后直接点击“小译同学”图标即可，无需电脑开机。

应用由免费的 GitHub Pages 提供静态文件，不包含后台服务。首次打开后会缓存练习页面，可离线继续使用；学习进度始终存储在 iPad 自己的 Safari 中。

完整图文说明见线上 [`guide.html`](https://tideonezpp.github.io/xiaoyi-english/guide.html)，应用左侧目录中也有“使用说明”入口。

如果只想在电脑本机使用，也仍可双击 `启动学习.bat`。

## 开发命令

```bash
npm install
npm run dev
npm run build
```

`npm run build` 会生成可直接由 GitHub Pages 发布的 `docs` 静态目录。

## 教材内容说明

目前的题目是按教材单元主题和核心句型原创整理的数据，并非教材原文扫描或整页复制。每个单元默认提供可直接点读的核心词句，不要求家长填写。要与课本逐字一致，需要以家庭持有教材的 PDF 或清晰照片生成内容包；教材原文不会放入公开 GitHub 仓库。
