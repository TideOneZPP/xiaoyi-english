# 小译同学 · iPad 静态课本点读

给家里小学生使用的译林版英语课本点读与练习工具。静态网站直接包含三至六年级上下册共 8 册、64 个单元、784 页教材数据，全部从已核验的教材 PDF 生成。

孩子在 iPad 打开网址即可选书、翻页、点读和练习：不需要电脑端、不需要导入教材包，也不需要家长复制课文。

## iPad 使用方法

1. 用 iPad Safari 打开 [小译同学](https://tideonezpp.github.io/xiaoyi-english/)。
2. 第一次等待教材目录和当前课页加载完成。
3. 点 Safari“共享”→“添加到主屏幕”。
4. 以后直接点击“小译同学”图标。

课页按需加载，不会在第一次打开时下载全部 154 MB。已经阅读过的部分页面会由 PWA 缓存，网络中断时仍可继续阅读缓存过的内容。

## 已实现功能

- 8 册课本直接显示在静态书架。
- 课本整页展示，横竖屏自动适配。
- 点击课页英文区域直接朗读。
- 长按选择英文后朗读。
- 优先选择 iPad 已安装的英文男声，并使用适合跟读的慢速、低音调和句间停顿。
- 点词跟读、听音辨句、原句补全、句子排序。
- 每道练习标注课本来源页码。
- `Mr.`、`Mrs.`、日期、时间和常用缩写播报预处理。

## 静态教材构建

家庭本地源数据保存在被 Git 忽略的 `textbooks/data/`。运行：

```bash
npm run build
```

构建过程先生成 PWA，再由 `scripts/export-static-textbooks.mjs` 将课页图片、OCR 点读坐标和单元练习数据复制到 `docs/textbooks/data/`。GitHub Pages 直接发布 `docs` 目录，不需要 API 或后端服务器。

## 内容说明

- 本项目仅供家庭非商业学习，与教材出版社及国家中小学智慧教育平台无隶属、授权或背书关系。
- 静态数据共约 155 MB，包括 784 张课页图片、784 份 OCR 文字层和 1 份课程目录。
- 自动练习仅选取高可信度课本英文，并过滤教学指令、人物名和明显 OCR 异常。
- 语音使用 iPad 自带的英文合成语音，不是出版社配套真人录音。
- 浏览器只保存上次阅读的册次与页码，不上传儿童资料或录音。
