# 小译同学 · 本地课本点读

给家里小学生使用的译林版英语课本点读与练习工具。当前接入三至六年级上下册共 8 册、64 个单元；课页、可点读文字、单词和练习原句全部从家庭本地保存的教材 PDF 生成，不再使用旧的原创课程数据。

孩子可以直接翻课本整页，点击页面上的英文朗读，也可以长按选择英文后朗读。右侧/页下提供四类课本来源练习：点词跟读、听音辨句、原句补全和句子排序，每道题都标注课本页码。

## iPad 家庭使用

1. 在电脑上双击 `启动学习.bat`。
2. 保持命令窗口和电脑开机。
3. 让 iPad 与电脑连接同一个 Wi-Fi。
4. 在 iPad Safari 打开命令窗口显示的地址，例如 `http://192.168.1.8:4173`。
5. 可从 Safari“共享”菜单选择“添加到主屏幕”。

这不是公网服务器：课本文件只由家里的电脑通过局域网提供。电脑关机、命令窗口关闭或 iPad 离开同一 Wi-Fi 后，iPad 将无法读取课页。

## 数据与隐私

- `textbooks/` 保存家庭本地 PDF、课页图片和 OCR 数据，已被 Git 忽略，不会上传 GitHub。
- GitHub 只保存程序、构建脚本和使用说明，不保存教材正文或课页图片。
- 工具没有账号、数据库、儿童资料上传或录音上传。
- 浏览器只保存上次阅读的册次与页码。
- 语音使用 iPad/浏览器自带的英文合成语音，不是出版社配套录音。

## 本地教材数据包

教材 PDF 放在 `textbooks/` 后运行：

```powershell
python scripts/build-textbook-data.py
```

脚本会生成 `textbooks/data/catalog.json`、每页图片和带坐标的 OCR 文字层。单元页码与标题的人工校对保存在本地 `textbooks/review-overrides.json`。自动练习只选取高可信度课本英文，并过滤目录、教学指令、页码和人物名等不适合作为独立练习的内容；整页点读仍保留 OCR 检出的英文。

## 开发命令

```bash
npm install
npm run dev -- --host 0.0.0.0 --port 4173
npm run build
```

`npm run build` 生成 `docs` 静态程序和 HTML 使用说明。公开 GitHub Pages 因不包含本地教材数据，只能显示数据准备提示；完整课本点读请使用上述局域网方式。

完整说明见 [`public/guide.html`](./public/guide.html)。
